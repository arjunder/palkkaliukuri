# Palkkaliukuri 2025

Kevyt ja moderni palkkalaskuri, joka arvioi nettopalkan ja veroprosentin Suomen vuoden 2025 veroperusteilla.  
Sovellus sisältää:

- Tumma ja vaalea teema
- Visuaalinen tuliefekti korkeille tuloille
- Ammattiesimerkit ja vertailukortit
- Selkeä UX ja reaktiivinen käyttöliittymä

## Käyttö

Avaa `index.html` selaimessa.  
Ei vaadi asennuksia, palvelinta tai kirjastoja.

## Rakenne

- **index.html** – koko sovellus HTML/CSS/JS yhdessä tiedostossa
- Ei ulkoisia riippuvuuksia
